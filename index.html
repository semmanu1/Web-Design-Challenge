Summary: Latitude vs. Other

The goal of this project was to study the patterns that occur in weather and how they correlate to the distnace from the equator. To complpete this analysis, data was pulled from OpenWeatherMap API and then compiled into one dataset with over 500 cities. After creating the dataset, utilizing Matplotlib, we studied various aspects of weather such as humidity vs latitude. This site provides an in depth look at all the data, the analysis and the visualizations to go with it.
